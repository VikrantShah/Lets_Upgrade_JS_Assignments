String methods

  1. charCodeAt() :

    2. charCodeAt() returns the Unicode value of a character present at the specified location.

    Example :
    let str = "Some random string";
    console.log(str);
    console.log(str.charCodeAt(5)); // 114

  2. search() :

    3. As the name suggests, it will look/search fot the specified regular expression in the given string and if the match occurs, the starting position of the string is returned.

    Example :
    let str = "Some random string";
    console.log(str.search("dom")); //8

  3. match() :

    4. match() will look/search fot the specified regular expression in the given string and if the match occurs, it will return the regular expression.

    Example :
    let str = "Some random string";
    console.log(str.match("dom"));

  4. replace() :

    5. replace() will replace a given string with some specified string and returns that string. Only the first found string will be replaced and rest all be intact. Original variable is not modified.

    Example :
    let str = "Some random string";
    console.log(str.replace("o", "a")); // Same random string

  5. substring() :

    6. It fetchs the string on the basis of provided index and returns the new substring. Doesn't make any changes to the original string.

    Example :
    let str = "Some random string";
    console.log(str.substring(5, 7)); // ra

  6. substr() :

    7. It fetchs the string on the basis of starting index & length. Returns the new substring. Doesn't make any changes to the original string.

    Example :
    let str = "Some random string";
    console.log(str.substr(1, 3)); // ome

  7. slice() :

    8. It fetchs the string on the basis of provided index and returns the new substring. Doesn't make any changes to the original string. It accepts positive as well as negative index.

    Example :
    let str = "Some random string";
    console.log(str.slice(1, -3)); // ome random str

  8. split() :

    9. As the name suggests, it splits a string into a substring array and returns the newly created substring array.

    Example :
    let str = "Some random string";
    console.log(str.split(" ")); // ["Some", "random", "string"]

  9. trim() :

    10. It trims the extra whitespaces from the left and right side of the string. Resultant string is also returned.

    Example :
    let str1 = " Some random string "
    console.log(str1.trim()); // Some random string

Array methods

  1. toString() :

    2. Converts an array into the string of array values. All the values are comma seperated.

    Example :
    let arr = [1, 2, 3, 4];
    console.log(arr); // [1, 2, 3, 4]
    console.log(arr.toString()); // 1,2,3,4

  2. join() :

    3. Converts an array into the string of array values. All the values are comma seperated. It acts as toString() but, seperator can also be specified.

    Example :
    let arr = [1, 2, 3, 4];
    console.log(arr.join(":")); // 1:2:3:4

  3. splice() :

    4. It is used to add multiple elements at the specified position. It also allows to specify, the number of elements to be removed while adding.

    Example :
    let arr = [1, 2, 3, 4];
    arr.splice(3, 0, 20, 30, 50);
    console.log(arr); // [1, 2, 3, 20, 30, 50, 4]

  4. slice() :

    5. It removes/slices out a piece of array into a brand new array. Doesn't remove any elements from the original array.

    Example :
    let arr = [1, 2, 3, 4];
    console.log(arr.slice(4, 6)); // Â [30, 50]
    console.log(arr); // [1, 2, 3, 20, 30, 50, 4]
