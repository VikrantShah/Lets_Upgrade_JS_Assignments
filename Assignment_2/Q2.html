Difference Between var, let and const

  1. var :

    2. The scope of the variable is global or local i.e. function scoped.
    The scope is global when, the var variable is declared outside and function/block of code. These variable is availabe to use in the whole code.
    The var variable declared within a function accessible with that function.
    If the variable is not declared and it is accessed, it gives "undefined".

    Example :
    console.log(declare); // Undefined
    var declare;

    function getCount()
    {
    var count;
    for(var i = 0; i < 10; i++)
    {
    var c = i;
    count = i - 1;
    }

    console.log(i);
    console.log(c);
    console.log(count);
    }
    getCount();
    console.log(count); // Reference Error

  2. let :

    3. var is also global scoped. Instead of being function scoped, let is block scoped. This means a variable created with let is available inside the block in which it is created as well as nested blocks.
    Block here means any code surrounded into curly braces {} like if statement, for loop, while loop, etc.
    If a variable is not declared and it is accessed, "ReferenceError" will be given.

    Example :
    console.log(declar); // Reference Error
    let declar;

    function getCount()
    {
    let count;
    for(let i = 0; i < 10; i++)
    {
    let c = i;
    count = i - 1;
    }

    console.log(i); // Reference Error
    console.log(c); // Reference Error
    console.log(count);
    }
    getCount();
    console.log(count); // Reference Error

    let pie = 3.14;
    pie = 22 / 7;

  3. const :

    4. const is same as let, only difference is that once the value is assigned, it cannot be changed.
    Once the value is assigned it cannot be changed.

    Example :
    const pi = 3.14;
    pi = 22 / 7; // Type Error
