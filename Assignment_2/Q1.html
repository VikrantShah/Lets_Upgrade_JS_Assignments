Methods of console object

  1. console.log() :

    2. console.log() is used to log/print/output a message on the console. It can display any form of data such as an array, string, boolean, etc.

    Example :
    console.log(1234);
    console.log("Some string");
    console.log([0, 1, 2]);
    console.log({1 : "xyz", 2 : "abc"});

  2. console.error() :

    3. console.error() is used for testing the code and display error messages. Error messages are displayed/highlighted in red color.

    Example :
    console.error("Some random error");

  3. console.warn() :

    4. console.warn() is used to display/log warnings in the console. Highlighted in yellow color.

    Example :
    console.warn("Some warning");

  4. console.clear() :

    5. As the name suggests, it is used to clear the console. In Chrome, “Console was cleared” message will be displayed after, clearing the console. In Firefox, no message is displayed after clearing the console.

    Example :
    console.clear();

  5. console.time() and console.timeEnd() :

    6. These functions are used when we want to know/calculate the time spend by a function or block of code to execute. Both functions take a label, which must be the same, and code inside it can be anything.

    Example :
    console.time();
    let func = function()
    {
    console.log("Function Running");
    }
    func();
    console.timeEnd();

  6. console.table() :

    7. Used to generate a table inside the console. Input must be an object/array.

    Example :
    console.table({1 : "xyz", 2 : "abc"});

  7. console.count() :

    8. console.count() will print the number of times it is called. A label can be added to the console view. By default, the label is “default”.

    Example :
    for(i = 0; i<=3; i++)
    {
    console.count();
    }
    // With some label
    for(i = 0; i<=3; i++)
    {
    console.count("Some Label");
    }

  8. console.group() and console.groupEnd() :

    9. These methods allows to group contents in a seperate intented block. These functions also accepts labels. Both functions take a label, which must be the same, and code inside it can be anything.

    Example :
    console.group("Group");
    console.log(1234);
    console.log("Some string");
    console.log([0, 1, 2]);
    console.log({1 : "xyz", 2 : "abc"});
    console.groupEnd("Group");
To see the Output of Example, press F12 on the keyboard.
