Datatypes in Javascript

  • Primitive Datatypes :-

    1. Number -

      2. Represents both integers and floating point numbers. Many operations can be performed on numbers, such as addition, subtraction, multiplication, etc.
      There are special numeric values that, also belong to this datatype such as "Infinity", "-Infinity" and "NaN".

    2. BigInt -

      3. In Javascript, the number datatype cannot represent positive numbers greater than (253 - 1) or the negative numbers less than -(253 - 1). This technical limitation is caused by some internal representation.
      Most of the purpose is fulfilled by this limit but, in some cases we require to work with really large numbers.
      BigInt was introduced to represent numbers of arbitary length. A BigInt is created by appending n to the end of integer.

    3. String -

      4. In Javascript, string is represented in the quotes. It can be either single or double qoutes i.e. If the string is started wkith single quote, it should end with single qoutes only. Same goes for double quotes.

    4. Boolean -

      5. Boolean or logical type can store only two values, either true or false.
      true means/represents yes or 1.
      false means/represents no or 0.

    5. null -

      6. This is a special/seperate type which doesn't matches with any other datatype. It represents "nothing", "empty" or "unknown value".

    6. undefined -

      7. This is also a special type of value/datatype which stands apart from any other value. It means the "any value is not assigned to it".
      If a variable is declared but, not assigned, then it's value is undefined.

  • Non-Primitive Datatypes :-

    1. Object -

      2. Object represents instance of a class through, which we can access the class members.

    2. Array -

      3. Array can store group of similar type of value. Multiple values can be stored in a single variable.

    3. RegExp -

      4. Regular Expression is a sequence of characters that forms a search pattern. Searching pattens can be used for text manipulation and text search. It can be single character or a very complex pattern.
